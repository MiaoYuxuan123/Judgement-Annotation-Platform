- 同一份数据可被再次分配给新增的标注者
- 一份案例与标注者之间为多对多的关系

- Q3：任务开始后，标签体系不变
- Q2：数据可能增加
- Q5：是
- Q1：任务开始后，可能出现人员更新

- Q8：序号动态更新，按照在原文本中的顺序生成
