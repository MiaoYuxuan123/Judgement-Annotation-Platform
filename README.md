# 裁判文书标注系统（Judgment Annotation Platform）

面向法律文本论证结构挖掘的可视化协同标注工具

## 项目介绍

本项目是一款**面向法学研究与教学场景**的裁判文书论证标注系统，支持命题抽取、标签体系标注、论证关系定义、自动生成论证图示，并提供多人双标注与冲突裁定能力，可用于法律论证挖掘数据集构建、模型训练与课堂实践。

## 核心功能

- 文本导入与裁判理由自动提取
- 三栏式标注界面（原文 + 标注区 + 论证图示）
- 命题选取、自动编号、标签体系（SF/GF/SM/GM）
- 论证关系标注（支持 / 反对 / 组合 / 匹配 / 同一）
- 实时论证图示生成与导出
- 多人独立标注 + 双结果对比 + 裁定终审
- 任务四阶段流程管理（创建→标注→裁定→导出）
- 数据导出（Excel/JSON/ 图片）

## 技术栈

- 前端：Web 浏览器端（Chrome/Edge）
- 后端：轻量化服务架构
- 导出格式：Excel、JSON、PNG/JPG/SVG
- 部署：支持校内服务器 / 本地运行

## 快速使用

1. 克隆仓库

```bash
git clone https://git.nju.edu.cn/44/sec-ii-2026.git
```

1. 部署启动服务
2. 登录系统 → 创建任务 → 上传文书 → 分配标注员
3. 标注员进行命题选取、标签与关系标注
4. 裁定者对比双标注结果并终审
5. 导出标注数据与论证图示

## 系统角色

- 超级管理员：用户管理、角色分配
- 任务创建者：上传文档、配置任务、分配标注人员
- 标注员：命题标注、关系标注、生成图示
- 裁定者：对比标注、冲突裁定、确认最终版本
